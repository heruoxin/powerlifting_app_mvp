# 首页.Tab01.信息页.训练容量热力图详情页

## 页面定位

该页面用于展开 `训练容量热力图` Widget。

首页 Widget 只展示压缩版热力图；本页负责把热力图放大，并提供更强的解释与分享能力。

## 进入方式

- 点击 `训练容量热力图` Widget 主体或 `查看详情` 进入。

## 页面目标

- 让用户清楚看到近期训练密度分布。
- 首版明确采用 GitHub 风格的每日训练图。
- 使用训练容量决定颜色明暗。
- 提供适合截图分享的直接分享动作。

## 页面结构

1. 顶部导航区
2. 模式切换区
3. 大图热力图区
4. 周摘要卡
5. 分享入口区

## 页面说明

### 模式切换区

- 首版不在本页切换频次模式。
- 支持按动作范围切换：
  - `三大项`
  - `全部动作`

### 大图热力图区

- 相比首页 Widget，这里使用更大的显示面积。
- 热力图下方要有一句明确图例说明，避免用户只觉得“好看”却看不懂。
- 首版设计明确对齐 GitHub 风格的每日训练图。
- 每个点对应一天，颜色深浅仅表示当日训练容量。

### 周摘要卡

- 总结该时间范围内最值得关注的结构信息。
- 可包含：
  - 每周训练天数
  - 平均密度
  - 最集中动作
  - 连续训练周数

### 分享入口区

- `长截图分享` 不进入下一级页面。
- 点击后，直接对当前页面执行长截图，并唤起系统分享面板。
- `查看相关训练` 跳去 `Tab02.计划页` 对应筛选态。

## 交互规则

- 切换模式时热力图区和摘要卡同时刷新。
- 点击某一个日期点，弹出当日训练摘要，不继续新建页面。
- 热力图配色要稳定，避免同一色深在不同模式下含义不一致。
