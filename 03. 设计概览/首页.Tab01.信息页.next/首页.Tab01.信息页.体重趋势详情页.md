# 首页.Tab01.信息页.体重趋势详情页

## 页面定位

该页面用于承接 `体重记录` Widget 的详情查看。

首页 Widget 负责快速输入，本页负责看趋势、看均值、看历史记录。

## 进入方式

- 点击 `体重记录` Widget 标题或主体区域进入。

## 页面结构

1. 顶部导航区
2. 当前体重与均值摘要卡
3. 体重趋势图
4. 快速录入卡
5. 历史记录列表

## 页面说明

### 当前体重与均值摘要卡

- 展示当前最近一次体重。
- 展示 `7 日均值` 和 `30 日均值`，避免用户只看单点波动。
- 显示相对上周的变化值。

### 体重趋势图

- 默认查看 `30 天`。
- 允许切换 `7 天 / 30 天 / 90 天`。
- 线图上可叠加一条移动平均线，但默认不应过于花哨。

### 快速录入卡

- 维持与首页 Widget 一致的低摩擦录入体验。
- 用户在这里录入并保存后，摘要卡和趋势图即时刷新。

### 历史记录列表

- 近期记录按时间倒序排列。
- `查看更多` 仅展开更多列表，不再单独新建页面。

## 交互规则

- 点击图上某个点，显示该日体重和相对均值偏差。
- 同一天允许多次记录，但默认只在摘要卡显示最新一条。
- 若用户录入异常值，应给出轻量确认，避免误触导致趋势失真。
