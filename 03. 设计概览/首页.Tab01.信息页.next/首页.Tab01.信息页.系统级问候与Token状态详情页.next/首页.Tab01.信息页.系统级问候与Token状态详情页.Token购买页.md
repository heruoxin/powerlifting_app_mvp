# 首页.Tab01.信息页.系统级问候与Token状态详情页.Token购买页

## 页面定位

该页面用于完成 `AI Token` 的购买。

它是一个高转化页面，但必须保持边界清晰：

- 强调 AI 能力付费。
- 不暗示基础训练记录功能会失效。
- 不使用模糊、诱导性的价格表达。

## 进入方式

- 从 `Token 状态详情页` 点击 `购买 Token` 进入。
- 从低余额提醒点击 `立即购买` 进入。

## 页面结构

1. 顶部导航区
2. 当前余额提示
3. 商品卡列表
4. 购买说明区
5. 恢复购买与协议入口

## 页面说明

### 商品卡

- 每张商品卡至少显示：
  - Token 数量
  - 价格
  - 使用场景建议
  - 选择按钮
- 允许平台标记一个 `推荐` 档位。
- 不使用倒计时、限时红点等强刺激促销语言作为首版默认策略。

### 购买说明区

- 明确说明：
  - 本地训练记录与笔记功能免费。
  - Token 仅用于 AI 对话、AI 总结、AI 分析等能力。
  - Token 消耗随任务长度和复杂度而变化。

### 恢复购买与协议入口

- `恢复购买` 用于原生内购恢复。
- `计费说明` 用于查看计费原则说明页或弹层。

## 交互规则

- 点击某个方案后，进入原生支付链路。
- 支付完成后，返回上一页并刷新余额。
- 支付失败或取消时，保留当前页面，不强制弹出二次劝购。
