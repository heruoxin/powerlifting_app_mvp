# 首页.Tab01.信息页.系统级问候与Token状态详情页

## 页面定位

该页面是信息页顶部固定系统级 `问候与 Token 状态卡` 主体区域的详情页。

它的目标不是承载完整账号中心，而是用最短路径回答以下问题：

1. 我现在还有多少 `AI Token`。
2. 我最近消耗得快不快。
3. 按当前速度大约还能用多久。
4. 我应该去哪里购买，或查看消耗明细。

## 进入方式

- 点击信息页顶部系统级卡片主体区域进入。
- 点击“预计还可使用 X 天”文本进入。
- 当余额较低时，也可从低余额提醒条进入。
- 系统卡右上角的 `设置` 按钮不进入本页，而是进入全局设置页。

不从这里处理的内容：

- 完整个人资料设置。
- 账号安全与登录管理。
- 与训练无关的泛账户中心功能。
- 全局设置页本身。

## 页面结构

1. 顶部导航区
2. Token 总览卡
3. 使用预测卡
4. 最近消耗来源卡
5. 操作入口区

## 页面说明

### 顶部导航区

- 标题固定为 `Token 状态详情`。
- 左侧 `返回` 回到信息页。
- 右侧 `帮助` 打开简短说明，解释：
  - `Token` 是什么。
  - 什么行为会消耗 Token。
  - “预计可用时长”只是预测值，不是承诺值。

### Token 总览卡

- 展示当前余额。
- 展示预测口径使用的时间窗口，例如 `近7日平均消耗`。
- 用横向条形图强化“剩余空间”感知。
- 当余额较低时，总览卡上方出现高优先级提醒条。

### 使用预测卡

- 默认使用 `近7日口径` 进行预测。
- 用户可切换到 `近30日口径`，查看更平滑的估算结果。
- 这里只展示趋势方向和大致剩余时长，不展示过细的数学细节。

### 最近消耗来源卡

- 按消耗场景聚合显示，例如 `聊天问答`、`训练总结`、`教练观察整理`。
- 只展示最近一段时间内的主要来源。
- 点击 `查看全部明细` 进入 `Token明细页`。

### 操作入口区

- `购买 Token` 进入 `Token购买页`。
- `消耗明细` 进入 `Token明细页`。

补充说明：

- 顶部系统级卡片本身固定置顶，且不参与普通 widget 排序。
- 该卡片右上角另有 `设置` 按钮，进入全局设置页；这条路径与本页平级，不属于本页下钻内容。

## 交互规则

- 点击总览卡主体不做二次跳转，避免把页面再套页面。
- 切换预测口径时，仅刷新预测卡，不刷新整页。
- 低余额提醒条始终固定在内容区顶部，直到用户余额恢复或主动关闭本次提醒。
