# 首页.Tab01.信息页.三大项PR与e1RM趋势详情页.单项目趋势页

## 页面定位

该页面是 `三大项 PR / e1RM 趋势详情页` 的单项目下钻页。

同一套页面结构可复用于：

- 深蹲
- 卧推
- 硬拉

## 页面目标

- 让用户更认真地看单个主项的变化。
- 把 `实际表现`、`e1RM`、`关键训练` 放到同一页对照。
- 提供适合截图分享的单项目展示视图。

## 页面说明

### 顶部摘要卡

- 标题按当前主项动态变化，例如 `卧推趋势`。
- 顶部卡展示：
  - 当前实际 PR
  - 当前 e1RM
  - 选定时间范围内的净变化

### 图表区

- 支持查看实际 PR 与 e1RM 的关系。
- 允许同一套时间范围筛选。

### 关键里程碑

- 只显示少量有意义节点，例如：
  - 首次突破某整数关口
  - 明显平台期结束
  - 当前周期内的新高

### 相关训练

- 列出最能解释当前趋势的几次关键训练。
- 点击 `查看训练记录` 跳去 `Tab02.计划页`。

## 交互规则

- 单项目页默认保留上一级页面的筛选状态。
- 若用户从分享友好角度浏览，本页要尽量减少杂乱按钮，保持顶部信息干净。
