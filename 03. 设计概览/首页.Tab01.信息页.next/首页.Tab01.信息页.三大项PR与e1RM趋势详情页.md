# 首页.Tab01.信息页.三大项PR与e1RM趋势详情页

## 页面定位

该页面用于展开 `三大项 PR / e1RM 趋势` Widget。

首页 Widget 只负责给出一个压缩后的趋势感知；本页负责让用户更认真地查看：

- 深蹲、卧推、硬拉的长期变化
- `实际 PR` 与 `推测 PR / e1RM` 的差异
- 某段时间内是否真的在进步

## 进入方式

- 点击信息页中的 `三大项 PR / e1RM 趋势` Widget 主体进入。

## 页面结构

1. 顶部导航区
2. 三大项摘要卡
3. 主图表区
4. 相关训练记录入口区

## 页面说明

### 三大项摘要卡

- 显示各主项的当前 `实际 PR`。
- 显示与之并列的当前 `e1RM`。
- 点击某项摘要卡进入 `单项目趋势页`。

### 主图表区

- 默认显示最近 `12 周`。
- 允许切换：
  - `12 周`
  - `24 周`
  - `52 周`
- 允许切换：
  - 只看 `实际 PR`
  - 只看 `推测 PR / e1RM`
  - `同时显示`

### 相关训练记录入口区

- 提供最近一次与该趋势强相关的训练入口。
- 点击后跳去 `Tab02.计划页` 对应训练详情。

## 交互规则

- 点击某个数据点，显示该次训练日期与对应训练对象。
- 点击某个主项摘要卡或筛选标签，进入或切换对应主项视图。
- 若某一主项近期无足够数据，显示空态说明，而不是画出误导性的平滑线。
